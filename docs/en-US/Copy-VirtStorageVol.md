# Copy-VirtStorageVol

## SYNOPSIS

Copy storage volume.

## SYNTAX

### Default (Default)

```
Copy-VirtStorageVol -Name <String> -Source <StorageVol> [-PreallocMetadata] [-RefLink]
 [-Server <Connection>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### __AllParameterSets

```
Copy-VirtStorageVol -Name <string> -Source <StorageVol> [-PreallocMetadata] [-RefLink]
 [-Server <Connection>] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

Copy storage volume.

## EXAMPLES

### Example 1

```powershell
PS C:\> # copy storage volume by specified key.
PS C:\> Get-VirtStorageVol -Key 'VolumeKey' | Copy-VirtStorageVol -Name 'VolumeName'
```
